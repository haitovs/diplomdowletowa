# Product Images for Dowletowa Textiles

Place all product images in: `public/images/products/`

## Image Naming Convention
Format: `product-{slug}-{number}.jpg`

Example:
- `product-tekke-desert-runner-1.jpg`
- `product-tekke-desert-runner-2.jpg`
- `product-ruby-gallery-carpet-1.jpg`

## Download Instructions

1. Download or create appropriate product images
2. Rename them according to the naming convention above
3. Place all images in `public/images/products/`
4. Images will automatically appear on product pages

## Image Specifications

- **Format**: JPG, PNG, or WebP
- **Recommended Size**: 1200x1200px minimum
- **Max File Size**: 2MB per image
- **Aspect Ratio**: Square (1:1) preferred

## Notes

- Images with `-1` suffix are the primary/main product image
- Multiple images per product allow for gallery views
- All images should showcase the product's craftsmanship and details
